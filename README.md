# WeToBee

- 프로젝트 사이트 -> **StubbyPlanner**
- 스투비 플래너는 유렵 여행을 보다 쉽게 플랜을 기획 할 수 있고 또한 전문가의 플랜을 이용 할 수있는 사이트 입니다.
- 위투비는 해당 사이트를 모티브하여 국내 여행 계획을 보다 쉽게 기획 할 수 있도록 도와주는 사이트입니다.
